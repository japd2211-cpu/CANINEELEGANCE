Cómo subir a tu dominio (opciones):
1) cPanel / Hosting tradicional:
   - Sube todo el contenido de esta carpeta al directorio público (generalmente public_html o htdocs).
   - Asegúrate de que el archivo principal se llame index.html.

2) Netlify (gratis y rápido):
   - Arrastra la carpeta al sitio app.netlify.com (New site from folder).
   - O conecta un repositorio y haz deploy continuo.

3) GitHub Pages:
   - Crea un repositorio, sube estos archivos a la rama main.
   - Activa Pages en Settings → Pages → Deploy from branch → / (root).

Personalización del logo:
- Reemplaza assets/logo.svg por tu logo real con el mismo nombre (o ajusta la ruta en index.html).
